# Automated Review Generator

Minimal starter plugin scaffold for 'Automated Review Generator' by Code Temple (https://codetemple.net).

Install:

```bash
# copy the folder into wp-content/plugins/
# then activate from WordPress admin
```

Verification:

1. Activate the plugin in WordPress Admin → Plugins.
2. Visit the plugin page in the admin sidebar: **Automated Reviews** → you should see a page that contains the text: `Admin page test.`
